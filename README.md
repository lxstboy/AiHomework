# AI作业求解器

一个简洁优雅的AI作业求解网站，支持批量上传图片并实时显示AI解答过程。

## 功能特点

- 🎨 **Apple风格设计** - 简洁优雅的用户界面
- 📸 **批量上传** - 支持一次上传多张作业图片
- 🤖 **AI智能解答** - 统一使用国内AI API
- 🌍 **智能语言切换** - 自动检测用户IP，根据地理位置显示中文或英文界面
- 📚 **多科目支持** - 数学、物理、化学、生物、语文、英语等
- 💭 **实时思考过程** - 显示AI的解题思路
- 💾 **结果下载** - 支持下载解答结果为文本文件

## 快速开始

### 安装依赖

```bash
npm install
```

### 配置API（可选）

系统已内置默认API配置，可以直接使用。如果需要使用自己的API密钥：

1. **创建 `.env` 文件**（在项目根目录）
2. **复制 `.env.example` 文件内容**，或直接创建包含以下内容：

```env
# API 基础地址（可选）
VITE_API_BASE_URL=https://api.suanli.cn/v1

# API 密钥（必填）
VITE_API_KEY=your_api_key_here
```

3. **填入您的API密钥**，保存文件
4. **重启开发服务器**（如果正在运行）

**注意**：
- `.env` 文件已添加到 `.gitignore`，不会被提交到Git
- 如果不配置，系统会使用内置的默认API密钥
- 修改 `.env` 后需要重启开发服务器才能生效

**语言自动切换**：
- 系统会自动检测用户IP地址
- 中文地区（中国大陆、台湾、香港、澳门、新加坡）→ 显示中文界面
- 其他地区 → 显示英文界面

### 运行开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **选择科目** - 在顶部选择对应的科目
2. **上传图片** - 点击上传区域或拖拽图片文件
3. **开始求解** - 点击单张图片的"求解"按钮，或使用"批量求解"
4. **查看解答** - 右侧面板实时显示AI的思考过程和解答
5. **下载结果** - 点击"下载"按钮保存解答结果

## 技术栈

- React 18
- TypeScript
- Vite
- OpenAI GPT-4 Vision API
- Lucide React Icons

## 注意事项

- 系统统一使用国内AI API，已内置配置
- 系统会自动检测用户IP地址并显示对应语言界面（中文/英文）
- 如果IP检测失败，默认使用中文界面
- 建议使用GPT-4 Vision模型以获得最佳效果
